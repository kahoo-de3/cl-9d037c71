# 開発記録（2026-07-16）

Claude Code とのセッションで作成。元データは `Book1.xlsx`（B=施設名, C=氏名, D=C, E=O、5〜108行・104件）。

## 経緯

1. **初版作成**: Excelを読み取り、単一HTMLのチェックリストアプリを生成。
   年(2026-2030)・月(1-12)プルダウン、C/Oタップで薄灰→濃灰の2段階グレーアウト、
   進捗バー、未完了フィルタ、localStorage自動保存。claude.aiアーティファクトとして公開
2. **氏名修正**: 104行目「校條　輔」→「校條　祐輔」。原因はExcel内の「祐」が
   外字U+E682（Webで表示不可）だったため。generate.py に自動置換を実装
3. **保存問題**: アーティファクト環境はサンドボックスが毎回作り直され、
   localStorageが再起動で消えることが判明（保存API自体は成功するため検出不能）。
   → GitHub Pages（本リポジトリ）に移行して解決。ランダムなリポジトリ名+noindexで検索避け
4. **PINロック追加**: ユーザー要望により、施設名・氏名データをAES-GCM暗号化して埋め込み。
   PIN→PBKDF2-SHA256(310k)で鍵導出。平文が含まれていた旧コミットは履歴書き換えで除去
5. **iOS Safariで解錠ボタンが押せない問題**: fixed配置+キーボードの当たり判定ずれと判断。
   ロック画面を通常配置に変更し、最終桁入力で自動解錠する方式に変更（v3）
6. **背景を白に固定**: ダークモードでも白（ユーザー要望）
7. **フィルタ不具合修正**: `.row{display:flex}` が hidden 属性を上書きして完了行が
   消えなかった → `[hidden]{display:none !important}` で解決
8. **バックアップ/復元ボタンを削除**（ユーザー要望）→ 完成

## 構成

```
docs/index.html   … アプリ本体（生成物・暗号化済み）。GitHub Pagesで配信
docs/robots.txt   … 検索避け
tools/generate.py … Excel → docs/index.html 生成スクリプト（唯一のソース）
README.md         … 仕様と差し替え手順
```

**アプリのHTML/CSS/JSはすべて `tools/generate.py` 内のテンプレートにある。**
デザインや挙動を変更するときは generate.py を編集して再生成する（index.htmlを直接編集しない）。
